SneakerBase

SneakerBase is an original interactive site that I created to manage and maintain my sneaker collection. It features a connection to Google Calendar to make notes on which shoes were worn by date. This project was challenging because it represents a huge passion of mine and therefore it was imperative to get it right. There is no existing sneaker API to call for the data for all 160+ shoes in my collection. This required scouring the Internet to compile the data for this project and then curating the gathered information into my database for each individual shoe.

To say the least, it was huge and daunting undertaking to collect all of the necessary information but the end result displays my passion in a very unique and exciting manner! The site is open to others to create and curate their own sneaker collections to form a community of Sneakerheads just like myself to share and manage their collections as well. It is a work in progress so there is still much to accomplish but as the product of an idea that started with a blank screen, I am very proud of what I have accomplished and of the final - but still unfinised - product!

Grovemade Clone

This was a group project with 3 teammates in which we made a clone of the website Grovemade.com. This was a challenging site to recreate, as it has won an Awwward (www.awwwards.com) for Design and honoring that was formidable. It was also difficult to build using an existing framework like Angular because each page of the original site is hard coded. Utilizing teamwork and a well-devised plan to manage the content, we were able to re-build the site in a modular manner making it much easier to update quickly for Grovemade, if they choose to utilize it! As a group, we decided upon the arrangement of the database schema. Once we decided upon that, I was responsible for creating and building the tables and then inserting the data into the database including nearly 600 pictures and the data for nearly 60 products and 10 differnt views. I also coded the Individual Item view as well as two of the Journal entries.

Tech Used:

Angular | JavaScript | Gulp | Sass | Stripe | Bootstrap | HTML/CSS | Node | Elephant SQL

The Original 25 Pokémon

Welcome to the world of Pokémon! This website focuses on the Original 25 Pokémon, from Bulbasaur to quite possibly the most famous Pokémon of all time, Pikachu. The data for this project was acquired from the PokéAPI with the exception of the pictures of each Pokémon. The pictures for each Sprite was garnered from an additional resource, the Bulbapedia. I then created my own JSON of these pictures and I built a second API call to this JSON to display the images in my project. This project was challenging but very satisfying, even with its simplistic subject matter. I built this for my 8-year old son who has shown it to many of his friends!

Tech Used:

Angular | JavaScript | HTML/CSS | JQuery | RESTful API
